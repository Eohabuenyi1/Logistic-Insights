# Logistic-Insights
This repository contains the Final Logistics Merge Dataset, a comprehensive record of logistics and shipment transactions. The dataset is designed to support analytics, performance tracking, and optimization studies within logistics and supply chain management domains.

The project provides a foundation for data-driven insights into areas such as:

Shipment performance evaluation

Cost optimization

Payment trends and delivery efficiency

Service type and regional distribution analysis

# Column Name  	    Description
SH_ID	            Unique shipment identifier
C_ID	            Customer identifier
SH_CONTENT	      Type of goods shipped (e.g., Healthcare, Electronics, Luggage)
SH_DOMAIN	        Shipment domain (Domestic / International)
SER_TYPE	        Service type (Regular / Express)
SH_WEIGHT	        Shipment weight (in grams)
SH_CHARGES	      Shipping charges associated with the order
M_ID	            Manager or handler ID
C_NAME	          Customer name
C_TYPE	          Customer category (Retail, Wholesale, Internal Goods)
Start_date	      Contract start date
End_date	        Contract end date
AMOUNT	          Total transaction amount
Payment_Status	  Indicates whether the payment is completed
Payment_Mode	    Mode of payment (e.g., Card Payment, COD)
Payment_Date	    Date of payment completion
Current_Status	  Shipment status (Delivered / Not Delivered)
Sent_date	Date    when shipment was sent
Delivery_date	    Date when shipment was delivered
E_BRANCH	        Branch or regional code for delivery
